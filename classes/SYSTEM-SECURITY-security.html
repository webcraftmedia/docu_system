Change logged in users Email.

change_email(stringnew_email, stringpost_script = null, stringpost_script_data = null) : bool
static

This will facilitate the @see \SYSTEM\TOKEN\token utility to generate a token and send it to the logged in users email using php mailinc function.

This function can only be invoked if the user is logged in and uses the function on himself.

This function will fail if the Email of the user is unconfirmed. You can only change the email of a confirmed account.
